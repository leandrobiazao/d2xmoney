# Project Setup Reminders

## Important Paths

### Frontend
- **Path**: `frontend/` (from project root)
- **Full Path**: `C:\App\angular\d2xmoney\frontend`
- **Port**: 4400
- **Start Command**: `cd frontend && npm start`

### Backend
- **Path**: `backend/` (from project root)
- **Full Path**: `C:\App\angular\d2xmoney\backend`
- **Port**: 8000
- **Virtual Environment**: `backend/venv/`
- **Activate Command (PowerShell)**: `cd backend && .\venv\Scripts\Activate.ps1`
- **Activate Command (CMD)**: `cd backend && .\venv\Scripts\activate.bat`
- **Start Command**: `python manage.py runserver`

## Critical Reminders

### ⚠️ ALWAYS Activate Python Virtual Environment Before Running Backend Commands

Before running any Python/Django commands, you MUST activate the virtual environment:

**PowerShell:**
```powershell
cd backend
.\venv\Scripts\Activate.ps1
```

**Command Prompt (CMD):**
```cmd
cd backend
.\venv\Scripts\activate.bat
```

### Common Commands

#### Frontend Commands (from `frontend/` directory)
```bash
cd frontend
npm start              # Start dev server
npm test              # Run unit tests
npm run e2e           # Run E2E tests
npm run e2e:ui        # Run E2E tests with UI
npm run build         # Build for production
```

#### Backend Commands (from `backend/` directory, AFTER activating venv)
```powershell
# Activate venv first!
.\venv\Scripts\Activate.ps1

# Then run commands:
python manage.py runserver           # Start Django server
python manage.py shell               # Open Django shell
python manage.py migrate              # Run migrations
python manage.py makemigrations       # Create migrations
```

#### Combined Commands (from project root)
```powershell
# Start both servers
cd frontend
npm run start:dev

# Or separately:
# Terminal 1 - Backend:
cd backend
.\venv\Scripts\Activate.ps1
python manage.py runserver

# Terminal 2 - Frontend:
cd frontend
npm start
```

## Running E2E Tests

**Prerequisites:**
1. Backend server must be running on port 8000
2. Frontend server will start automatically via Playwright webServer config

**Command:**
```powershell
cd frontend
npm run e2e
```

**Note:** The Playwright config automatically starts the frontend server, but the backend must be running separately.
